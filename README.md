1️⃣ What is the difference between var, let, and const?
  Answer - var is a scoped function and allow to re-declare, which can be a bug. let and const are block-scoped variable.  use let for variables that will changeble and const for values that are unique. 
2️⃣ What is the spread operator (...)?
  Answer - The spread operator allows you to quickly copy or combined array and objects into new ones. It spreads the individual elements of an iterable into a list of elements.
3️⃣ What is the difference between map(), filter(), and forEach()?
  Answer - forEach executes a function for each element, while map creates a new array by transforming every item. filter also creates a new array, but it only includes element that pass a specific condition.
4️⃣ What is an arrow function?
  Answer - An arrow function is a compacting way to write a function using the => syntax. They are great for short logic and do not have their own this context, make them perfect for callbacks.
5️⃣ What are template literals?
  Answer - Template literals allow you to create strings using backticks (`) instead of quote. They making it easy to insert variable directly into text using ${variableName} and support multi-line string.
